# Manasvi - A Mental Health Chatbot

**Manasvi** is an AI-powered chatbot designed to provide emotional support to individuals. It uses natural language processing (NLP) to engage in meaningful conversations, assess the emotional tone of user input, and guide users toward emotional well-being.

## Features
- **Sentiment Analysis**: Detects the emotional tone of messages (positive, neutral, or negative) to tailor responses accordingly.
- **Dynamic Conversations**: Engages users with personalized, context-aware conversation flows based on their inputs.
- **Real-time Emotional Support**: Offers immediate support to users 24/7.
- **Confidential and Anonymous**: Ensures a secure and private space for users to express their thoughts and emotions.
- **Feedback Mechanism**: Users can rate their experience and provide feedback to help improve the chatbot.

## Technologies Used
- **Django**: Backend framework for handling server-side logic and user interactions.
- **spaCy**: NLP library used to process user inputs and perform sentiment analysis.
- **TextBlob**: Additional sentiment analysis tool for emotional tone detection.
- **jQuery/JavaScript**: Used for dynamic interaction on the front end.
- **HTML/CSS**: For designing the chatbot's user interface.
